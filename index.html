Bug Bounty

Bug Bounty is an application designed for business level and development level users to report and track bugs on the different internal applications we use within Revature. Bug Bounty is unique in that it provides gamification of the bug fixing process through a score associated with resolving bugs. This will allow for more inherent competitiveness throughout the process or bug fixing across Revature's suite of enterprise level applications.

Roles / Responsibilities

  • Developed the Entity Relationship Diagram and implemented it in PostgreSQL for the team.
  • Performed as an integral part of a team to create and implement Test Strategies, Plans, and Cases.
  • Built key features of the program end to end using Angular, Spring, and SQL.
  • Implemented Test Driven Development concepts to write test cases for features of the program.
  • Included automation frameworks, such as Selenium and Protractor, to test our user interface.
  • Incorporated Behavior Driven Development concepts, primarily using Cucumber, to test the functionality of our user interface.
  • Strived to create an intuitive, pleasing user experience in the Angular front end.
  • Tracked progress and coordinated with other teams using Asana and other user-story tracking applications.

Environment / Technologies

Java, JavaScript, TypeScript, Spring, Angular, Protractor, Selenium / Webdriver, JUnit, TestNG, Jasmine, Jenkins
