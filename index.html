Bug Bounty

Bug Bounty is an application designed for business level and development level users to report and track bugs on the different internal applications we use within Revature. Bug Bounty is unique in that it provides gamification of the bug fixing process through a score associated with resolving bugs. This will allow for more inherent competitiveness throughout the process or bug fixing across Revatures suite of enterprise level applications.

The Fitness App

The Fitness App is an application that assists in the user's fitness goals. This application will allow the user to traverse through a simple, clean user interface, generate a workout plan and view their results. The user can use pre-made workout regiments or create their own daily plan that tracks their exercise and repetitions, allowing them to track their progress throughout use of the application. Other features include sharing workout regiments with others, generating a workout plan based on the user's body type and fitness goals, generating specific workouts based on a body area, and a weight tracker.

TRMS
